# TypeScript Basics - Coding Examples & Exercises

This repository contains my personal coding practices, exercises, and examples from the book **"TypeScript Basics" (Основы TypeScript / Pro TypeScript)** by Adam Freeman.

The goal of this repository is to demonstrate a solid understanding of TypeScript fundamentals, its type system, advanced features, and how it integrates with modern JavaScript development workflows.

## 🚀 Repository Structure

The code is structured into modules and concepts following the logical progression of Adam Freeman's book:

```text
.
├── README.md
├── 01-introduction/                # Setting up the environment, compiler (tsc), and first app
├── 02-understanding-types/         # Primitive types, type inference, type annotations, and type guards
├── 03-functions/                   # Function types, optional/default parameters, rest parameters, and overloading
├── 04-arrays-tuples-enums/         # Working with collections, strictly typed tuples, and enums
├── 05-objects-interfaces/          # Defining object shapes, optional properties, interfaces, and type aliases
├── 06-classes-inheritance/         # OOP in TypeScript: classes, access modifiers (public, private, protected), and abstract classes
├── 07-generics/                    # Generic functions, generic interfaces, classes, and generic constraints
├── 08-advanced-types/              # Union and intersection types, literal types, index signatures, and mapped types
├── 09-modules-namespaces/          # Organizing code with ES modules and namespaces
└── 10-typescript-in-practice/      # Configuring tsconfig.json, linting, and building for production
